# 商业协作

这里是从开源学习路径进入 To-B 协作的公开入口。

这个仓库本身帮助你理解和复现：

- `local-v2` 入门 demo
- 最小 FastAPI 服务
- 官方 `ESP-VoCat v1.2` hardware baseline
- `v3` public-preview 进阶路径

当你的团队需要把 demo 变成可交付产品时，就可以进入商业协作范围。

## 适合商业协作的情况

- 你需要生产部署或私有化部署。
- 你有自己的板卡，需要适配 MCU、codec、麦克风或扬声器链路。
- 你希望把低延迟语音交互做进真实产品。
- 你需要定制音色、人格、交互体验或业务流程。
- 你需要长期维护、迭代和交付支持。

## 当前联系路径

当前公开联系入口：

- 在本仓库 GitHub Issues 中提交 inquiry。
- 尽量附上 `commercial/intake-form.md` 中的问题答案，方便第一次回复更具体。

如果你不想公开敏感信息，可以只在 issue 中留下简短项目摘要，并说明希望后续走私下沟通。
