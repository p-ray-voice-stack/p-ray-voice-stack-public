# P-Ray Voice Stack

> **P-Ray Voice Stack:** `Let hardware products speak, and let everything come alive.`<br>
> Through transparent, privatizable open-source tutorials, we make it easy for all kinds of hardware products (e.g. toys, statues, and more) to access cutting-edge AI voice interaction capabilities.

> **Commercial solutions available:** 24/7 cloud deployment and maintenance, embedded support, professional knowledge base customization, voice cloning, and character/IP customization.

<br>


## Tutorials And Learning Paths

**Beginner Path**: PCM audio serial (10s-15s latency: **extremely simplified**)
- Software-only version: **local-v2** (10-minute quick start - runs locally in the browser)
- Hardware version: **hardware-v2**

<br>

**Advanced Path**: PCM uplink + Opus streaming downlink (3-5s latency: **initially product-grade**)
- Hardware version: **hardware-v3**

<br>


**Future Path (To-do)**
- Version-v4: **fast half-duplex switching (simulated full duplex)** - speech language model
- Version-v5: **native full duplex** - voice
- Version-v6: **multimodal full duplex** - image + voice + text
