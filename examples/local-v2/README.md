# Local V2

`local-v2` is the default first entry for this repository. It is a minimal pure-software browser demo for developers who want to understand the v2 flow before touching hardware.

## What You Get

- a static browser page that simulates one local-v2 interaction
- zero hardware requirements
- a visible placeholder for answer audio in the UI

## What Is Not Included Yet

- real microphone capture
- real speech-to-text or text-to-speech
- streaming audio or v3 transport behavior

## Run

1. Open `examples/local-v2/web/index.html` in a browser. No server is required for this demo.
2. Click `Run Local V2 Demo`
3. Read the simulated user turn, assistant reply, and current placeholder audio note

## Why This Exists

This page gives a new developer a concrete first screen and interaction model without forcing them into hardware setup, microphone permissions, or cloud deployment on day one.
